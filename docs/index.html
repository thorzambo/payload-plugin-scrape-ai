payload-plugin-scrape-ai

Make any Payload CMS website instantly accessible to AI agents and LLMs.

Install Now View on GitHub

Features

SEO, but for AI.

AI agents can't parse your HTML efficiently. This plugin auto-generates clean, structured content in every format they understand.

☰

llms.txt Standard

Generates /llms.txt and /llms-full.txt following the official spec. The entry point AI agents look for.

↻

Auto-Sync

Every content change triggers instant regeneration. No cron jobs, no manual triggers. Hybrid event + queue architecture handles bulk edits gracefully.

⚙

AI Enrichment

Optional AI-powered summaries, entity extraction, topic classification, and semantic chunking for RAG pipelines. Works without AI too.

{}

JSON-LD

Schema.org structured data for every page. Products, articles, services — automatically mapped from your collection types.

★

Smart Detection

Auto-discovers which collections contain content worth exposing. Pages, posts, products detected automatically. Toggle from the dashboard.

$

Token Estimator

Estimates total AI cost before you enable enrichment. Recommends the cheapest model that fits your content. No surprise bills.

API

Public Endpoints

All endpoints are public by design — AI agents need direct access. Rate-limited per IP.

Endpoint Type Description
GET /api/llms.txt text/markdown Curated content index
GET /api/llms-full.txt text/markdown Complete content listing
GET /api/ai/:collection/:slug.md text/markdown Individual page markdown
GET /api/ai/sitemap.json application/json Content graph & hierarchy
GET /api/ai/structured/:collection/:slug.json application/json JSON-LD structured data
GET /api/ai/context?query=...&limit=5 application/json Relevance-scored search
Get Started

Installation

1. Install the package

npm # From GitHub (current) npm install github:thorzambo/payload-plugin-scrape-ai # From npm (coming soon) # npm install @thorzambo/payload-plugin-scrape-ai

2. Add to your Payload config

// payload.config.ts import { scrapeAiPlugin } from 'payload-plugin-scrape-ai' export default buildConfig({ plugins: [ scrapeAiPlugin({ siteUrl: 'https://your-website.com', siteName: 'My Website', siteDescription: 'What this site is about', }), ], })

3. That's it. Visit /admin/scrape-ai to see the dashboard.
